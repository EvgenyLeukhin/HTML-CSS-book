  </div>
</div> 		В Bootstrap 12-колоночный тип сеток
ВЛОЖЕННОСТЬ Можно вкладывать .row внутрь .row 
<div class="container">
  <div class="row">

    <div class="col-...">
      <div class="row">
        <div class="col-..."></div>

      </div>
    </div>
  </div>

КЛАССЫ

КАК РАБОТАТЬ С КЛАССАМИ:

В html-коде присваиваем класс сеток (.col-lg, .col-md и т.д.), а потом добавляем кастомный класс для стилизации (.page-header, .content-box и т.д.) и стилизуем этот класс.

Не трогаем классы бутстрапа и не изменяем bootstrap.css !

<div class="main-content  col-lg-12 ..."></div>

container          - ОСНОВНОЙ контейнер (width: размеру breakpoints;)
container-fluid    - ОСНОВНОЙ контейнер резиновый (width: 100%;)

row                - СТРОЧНЫЙ подконтейнер, объединяет кол. классы 
row-fluid          - СТРОЧНЫЙ подконтейнер c (padding: 0 15px;)

col                - КОЛОНОЧНЫЙ под-контейнер
col-lg-*           - КОЛОНОЧНЫЙ под-контейнер *-колонок для lg
col-md-*           - КОЛОНОЧНЫЙ под-контейнер *-колонок для md
col-sm-*           - КОЛОНОЧНЫЙ под-контейнер *-колонок для sm
col-xs-*           - КОЛОНОЧНЫЙ под-контейнер *-колонок для xs

col-lg-offset-*    - Отступ от контейнера вправо на *-колонок для lg (доп. класс)
col-md-offset-*    - Отступ от контейнера вправо на *-колонок для md (доп. класс)
col-sm-offset-*    - Отступ от контейнера вправо на *-колонок для sm (доп. класс)
col-xs-offset-*    - Отступ от контейнера вправо на *-колонок для xs (доп. класс)

col-md-push-*      - Толкать вправо на *-колонок для md (доп. класс)
col-md-pull-*      - Толкать влево на *-колонок для md (доп. класс)

visible-xs         - Будет отображаться при xs
hidden-xs          - Будет скрыт при xs

3. Навигация
Меню 
<!--КОНТЕЙНЕРЫ-->
<div class="container-fluid">
  <div class="row">
    <!--БЛОК НАВИГАЦИИ (inverse или default)-->
    <div class="navbar navbar-inverse">
      <!--КОНТЕЙНЕР-->
      <div class="container">
        <!--ЗАГОЛОВОК НАВИГАЦИИ-->
        <div class="navbar-header">
          <!--ССЫЛКА НА КОМПАНИЮ-->
          <a href="#" class="navbar-brand">Компания</a>
          <!--КНОПКА ВЫП. МЕНЮ-->
          <button 
            class="navbar-toggle"
            type="button" 
            data-toggle="collapse"
            data-target="#responsive-menu"
          >
            <span class="sr-only">Открыть навигацию</span>
            <span class="icon-bar"></span>
            <span class="icon-bar"></span>
            <span class="icon-bar"></span>
          </button>
        </div>
        <!--ПУНКТЫ МЕНЮ-->
        <div class="collapse navbar-collapse" id="responsive-menu">
          <ul class="nav navbar-nav">
            <li><a href="#">Пункт1</a></li>
            <li><a href="#">Пункт2</a></li>
            <li><a href="#">Пункт3</a></li>
            <li><a href="#">Пункт4</a></li>
            <li><a href="#">Пункт5</a></li>
            <li><a href="#">Пункт6</a></li>
            <li>
              <a href="#">Пункт7
                <b class="caret"></b>
              </a>
            </li>
          </ul>
        </div>
      </div>
    </div>
  </div>
</div>
navbar - Блок навигации
navbar-default - Блок навигации по ум* (серая)
navbar-inverse - Блок навигации по ум* (чёрная)
navbar-header - Блок заголовка навинации
navbar-collapse - Блок меню
navbar-fixed-top - Блок меню зафикс. и растянутый
navbar-fixed-bottom - Блок меню зафикс. и растянутый снизу
navbar-static-top - Блок меню зафикс. и растянутый снизу
navbar-static-bottom - Блок меню зафикс. и растянутый снизу
caret - Выпадающее подменю
Классы header>nav.navbar.navbar-default>div.container

  • navbar - основной класс для навигации
  • navbar-default - подкласс (серый цвет)
  • navbar-inverse - подкласс (чёрный цвет)
  • container - контейнер для содержимого
