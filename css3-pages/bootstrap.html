1. Подключение Bootstrap
HTML-код 

<html lang="ru">
  <head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <!-- The above 3 meta tags *must* come first in the head; any other head 4
    content must come *after* these tags -->
    <title>Bootstrap 101 Template</title>
    <!-- Bootstrap -->
    <link href="css/bootstrap.min.css" rel="stylesheet">
    <!-- HTML5 shim and Respond.js for IE8 support of HTML5 elements and media queries -->
    <!-- WARNING: Respond.js doesn't work if you view the page via file:// -->
    <!--[if lt IE 9]>
      <script src="https://oss.maxcdn.com/html5shiv/3.7.3/html5shiv.min.js"></script>
      <script src="https://oss.maxcdn.com/respond/1.4.2/respond.min.js"></script>
    <![endif]-->
  </head>
  <body>
    <h1>Hello, world!</h1>
    
    <!-- jQuery (necessary for Bootstrap's JavaScript plugins) -->
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>
    <!-- Include all compiled plugins (below), or include individual files as needed -->
    <script src="js/bootstrap.min.js"></script>
  </body>
</html>
Условные комментарии можно в принципе удалить, поддержка IE8 уже почти не испльзуется

2. Сетки на Bootstrap
КОНТЕЙНЕРЫ <div class="container">
  <div class="row"> <!--Имеет 12 колонок-->

    <div class="col-lg-6 col-md-5 col-sm-4 col-xs-3">     </div>

    <div class="col-lg-6 col-md-7 col-sm-8 col-xs-9">     </div>

  </div>
</div>
Внутри колонок могут быть еще вложенные дивы с классом row
Классы container\container-fluid имеют padding: 15px; по умолчанию 		В Bootstrap 12-колоночный тип сеток
.container - Общий контейнер
.container-fluid - Общий контейнер (Резиновый, 100%)
.row - Подконтейнер-строка
.col - Подконтейнер-колонка

Значения ширины по умолчанию



3. Классы колонок
Виды классов <div class="col-lg-6 col-lg-offset-2">
<div class="col-lg-4 hidden-sm">
<div class="col-md-4 hidden-sm visible-lg"> 		col-lg-offset-2 - Отступ на 2 колонки от левого края при данной ширине дисплея, по ум* 0
hidden-sm - Колонка станет невидимой при данной ширине дисплея
visible-lg - Колонка станет видимой при данной ширине дисплея

4. Навигация
Меню 
<!--КОНТЕЙНЕРЫ-->
<div class="container-fluid">
  <div class="row">
    <!--БЛОК НАВИГАЦИИ (inverse или default)-->
    <div class="navbar navbar-inverse">
      <!--КОНТЕЙНЕР-->
      <div class="container">
        <!--ЗАГОЛОВОК НАВИГАЦИИ-->
        <div class="navbar-header">
          <!--ССЫЛКА НА КОМПАНИЮ-->
          <a href="#" class="navbar-brand">Компания</a>
          <!--КНОПКА ВЫП. МЕНЮ-->
          <button 
            class="navbar-toggle"
            type="button" 
            data-toggle="collapse"
            data-target="#responsive-menu"
          >
            <span class="sr-only">Открыть навигацию</span>
            <span class="icon-bar"></span>
            <span class="icon-bar"></span>
            <span class="icon-bar"></span>
          </button>
        </div>
        <!--ПУНКТЫ МЕНЮ-->
        <div class="collapse navbar-collapse" id="responsive-menu">
          <ul class="nav navbar-nav">
            <li><a href="#">Пункт1</a></li>
            <li><a href="#">Пункт2</a></li>
            <li><a href="#">Пункт3</a></li>
            <li><a href="#">Пункт4</a></li>
            <li><a href="#">Пункт5</a></li>
            <li><a href="#">Пункт6</a></li>
            <li>
              <a href="#">Пункт7
                <b class="caret"></b>
              </a>
            </li>
          </ul>
        </div>
      </div>
    </div>
  </div>
</div>
navbar - Блок навигации
navbar-default - Блок навигации по ум* (серая)
navbar-inverse - Блок навигации по ум* (чёрная)
navbar-header - Блок заголовка навинации
navbar-collapse - Блок меню
navbar-fixed-top - Блок меню зафикс. и растянутый
navbar-fixed-bottom - Блок меню зафикс. и растянутый снизу
navbar-static-top - Блок меню зафикс. и растянутый снизу
navbar-static-bottom - Блок меню зафикс. и растянутый снизу
caret - Выпадающее подменю
