box-shadow
Тень от блока
box-shadow: inset 1px 2px 3px 4px #ccc;
inset - |внутренняя|
1px - |смещение ох| + -
2px - |смещение оy|+ -
3px - |радиус размытия|+
4px - |растяжение| + -
#ccc - |цвет| 		.some-class {
    box-shadow: 5px 5px 1px 2px deeppink;
}
Внутрення тень (inset) .some-class {
    box-shadow: inset 5px 5px 0px 0px red;
}
Полупрозрачная тень (inset) .some-class {
    box-shadow: -7px -7px 2px 5px
    rgba(12, 256, 32, 0.5);
}
Множественная тень, перечисляются через запятую .some-class {
    box-shadow:
        4px 4px 1px 0 red,
        8px 8px 1px 0 green,
        12px 12px 1px 0 blue;
        16px 16px 1px 0 yellow;
}
