Мой Справочник | CSS3 | Таблицы на CSS3 | inline-таблица

display: inline-table;
Инлайн-таблица. Ведёт себя в потоке как inline-block, при этом сохраняет все табличные свойства и не переносится на новую строку 
        
<div class="inline-table"> 
 <div class="row">
  <div class="cell">1-1</div>
  <div class="cell">1-2</div>
  <div class="cell">1-3</div>
 </div>
 <div class="row">
  <div class="cell">2-1</div>
  <div class="cell">2-2</div>
  <div class="cell">2-3</div>
 </div>
</div>

        
.inline-table {
    display: inline-table;
    border-collapse: separate;
    border-spacing: 5px 10px;
}
.row {display: table-row;}
.cell {
    display: table-cell;
    height: 50px;
    width: 30px;
    border: 1px solid black;
    text-align: center;
    vertical-align: middle;
}
