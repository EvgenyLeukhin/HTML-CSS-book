Название цвета словом

color: maroon;
color: deeppink;
color: lightblue;

В настоящее время существует 17 одобренных имен, обозначающих цвета: aqua, black, blue, fuchsia, gray, green, lime, maroon, navy, olive, orange, purple, red, silver, teal, white и yellow

16-тиричный формат (0-9, a-f)

color: #1198f5;
color: #62f808;
color: #0af5c6;

RGB-формат (3 канала)

color: rgb(56, 140, 123);
color: rgb(30%, 40%, 50%);
color: rgb(145, 16, 100);

RGBA-формат (+ альфа-канал)

color: rgba(56, 140, 123, 0.85);
color: rgba(50%, 40%, 100%, 0.55);
color: rgba(145, 16, 100, 0.35);

HSL-формат
(1 - поворот по цветовому колесу,
2 - насыщенность,
3 - яркость)

color: hsl(0, 50%, 50%);
color: hsl(0, 75%, 75%);
color: hsl(0, 100%, 50%);

HSLA-формат (+ альфа-канал)

color: hsl(0, 100%, 50%, 0.5);
color: hsl(0, 100%, 50%, 0.75);
color: hsl(0, 100%, 50%, 1.0);

Прозрачный цвет

color: transparent;

Сокращенная 16-ая запись

color: #ff0000; = color: #f00;

Opacity - прозрачность

    background-color: red;
    opacity: 1;
    background-color: red;
    opacity: 0.75;
    background-color: red;
    opacity: 0.5;
    background-color: red;
    opacity: 0.25;

Запись цвета для кроссбраузерности:

color: rgb(255,100,50); /* Для IE8 */
color: rgba(255,100,50,.5); /* Для более современных браузеров */

Чтобы перестраховаться на случай неподдержки полупрозрачности или rgb-формата:
background: #f5f5f5;
background: rgba(245, 245, 245, 0.9);

Чтобы IE понимал прозрачность:

<!--[if IE]>
    <style type="text/css">
    .color-block {
        background:transparent;
        filter:progid:DXImageTransform.Microsoft.gradient(startColorstr=#50990000,endColorstr=#50990000);
        zoom: 1;
    }
    </style>
<![endif]-->

сurrentColor

При применении свойства color, автоматически происходит сохранение этого значения в переменную сurrentColor, и можно записывать css-код таким образом, если нужно присвоить тоже значение цвета.

div {
  color: #888;
  border: 1px solid currentColor;
  ... или так ...
  border: 1px solid;
}

При такой записи вернётся значение того же цвета, что и у color
