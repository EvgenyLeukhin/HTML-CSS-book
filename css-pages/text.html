Добавление нестандартных шрифтов

I способ - Веб-шрифты (через <link>)

1. Копируем ссылку из Google Fonts или любого другого ресурса на расположение шрифта и вставляем её в <head>

<link href='https://fonts.googleapis.com/css?family=Indie+Flower' rel='stylesheet' type='text/css'>

2. Копируем с сайта строку font-family в CSS и готово;

font-family: 'Indie Flower', cursive; (НАСЛЕДУЕТСЯ!)
<span style="font-family: 'Indie Flower', cursive;">Some text</span>
Some text

II способ (через @font-face)

1. Скачиваем шрифт с Google Fonts, Webfont, Fontriver

2. Подключаем с помощью к CSS-файлу@font-face (CSS-код может быть вместе со скаченным шрифтом, нужно вставить на самый верх CSS-кода)

Сейчас(2017) уже достаточно загружать 2 формата (woff2 и woff), форматы eot, ttf, svg загружать больше нет смысла и у них большой вес и поддержка старых версий браузеров
@font-face {
  font-family: 'Anfisa Grotesk';
  src: local('Anfisa Grotesk'), local('Anfisa_Grotesk'), local('Anfisa-Grotesk'),
       url('../fonts/anfisa/anfisagrotesk.woff2') format('woff2'),
       url('../fonts/anfisa/anfisagrotesk.woff') format('woff');
  font-weight: normal;
  font-style: normal;
}

Можно подключать различные варианты шрифта, указывая одно название font-face для всех, изменяя только font-weight и font-style, а применять также с одним названием, изменяя только fw и fs, нужный шрифт будет подгружаться

Лучше по максимуму использовать системные шрифты, их уже достаточно много в каждой ОС

Иконочный шрифт

1. Создаём свой иконочный шрифт на Glyphter и скачиваем его;
2. Копируем CSS-код в созданном сервисом css-файле и вставляем его.
3. font-family: "Glyphter";

Печатаем в коде: A B C D E F
Получаем в браузере: A B C D E F

Добавлять иконки с помощью иконочных шрифтов уже считается плохим тоном (большой вес шрифта + доп. запрос), лучше использовать svg-спрайты!!!

Как задавать font-size

Пиксели (Фиксированное значение)
font-size: 10px;

Просто и понятно. Фиксированное абсолютное значение

em (Отн. значение от родителя)
body { font-size: 20px; }
h1 { font-size: 2em; } // 40px
h1 span { font-size: 1em; } // 40px

em — в типографике относительная единица длины, равная размеру текущего (унаследованного шрифта) (размер буквы M, напечатанной шрифтом Cicero). Базовая 1em = 16px;, но она переопределяется для дочерних элементов.
Если вы прописали для заголовка (h1) font-size: 2em, и если у него родительский элемент body c font-size: 1em, то вычисляем для h2 font-size=16*2=32px, но если у h1 есть вложенный элемент span, то уже для этого span font-size: 1em уже будет равен 32px, а 2em = 64px

rem (root em, отн. значение от корня)
:root { font-size: 16px; }
h2 { font-size: 2rem; } // 32px
h2 span { font-size: 1rem; } // 16px

rem - по сути тоже самое, что em, только значение размера шрифта зависит не от размера родительского элемента, а от размера корневого элемента (:root) и не бужет изменяться от вложенности

% (тоже самое, что и em, только запись в процентах)
body { font-size: 20px; }
h1 { font-size: 200%; } // 40px
h1 span { font-size: 100%; } // 40px
vw (Проценты от ширины viewport)
h1 { font-size: 10vw; }

В примере: 10 процентов от высоты вьюпорта. Адаптивное значение

Как задавать line-height

Пиксели (Самое распространённое)
line-height: 20px;

Просто и понятно. Фиксированное абсолютное значение

Множитель (Адаптивное значение)
font-size: 20px;
line-height: 1.5;

Зависит от размера шрифта. line-height = 20px * 1.5 = 30px;

Проценты (Адаптивное значение)
font-size: 20px;
line-height: 150%;

То же самое как и множитель, только в процентах. line-height = 20px * 1.5 = 30px;

Ключевое слово
font-size: 20px;
line-height: normal;

Будет вычисляться автоматически по стандартным свойствам браузера

Несколько шрифтов:

font-family: "PT Sans", Arial, serif;
Будет грузится следущий через запятую шрифт, если нет первого
serif - шрифт с засечками, для больших частей текста, более читабельный (Times, Times New Roman, Georgia);
sans-serif - без засечек, для заголовков (Arial, Helvetica и Verdana);
monospace - моноширинный, каждый символ одинаковой ширины, для кода (Courier, Lucida Console и Monaco);

Если название шрифта состоит из нескольких слов, вам следует заключать его в кавычки!!!
font-family: "Times New Roman", Times, serif;

Форматы шрифтов

  • EOT - только в IE;
  • TTF(True Type) и OTF(Open Type) - эти типы шрифтов имеют широкую поддержку в большинстве веб-браузеров;
  • WOFF - Web Open Font Format представляют собой самый новый формат шрифта, разработанный специально для веб-технологий;
  • SVG - Scalable Vector Graphic формат (масштабируемой векторной графики) сам по себе не является форматом шрифта.

Чтобы гарантировать предоставление шрифтов как можно большему количеству браузеров, нужно как минимум использовать шрифты форматов EOT, WOFF и TrueType.
