Добавление нестандартных шрифтов

I способ - Веб-шрифты (через <link>)

1. Копируем ссылку из Google Fonts на расположение шрифта и вставляем её в <head>

<link href='https://fonts.googleapis.com/css?family=Indie+Flower' rel='stylesheet' type='text/css'>

2. Копируем с сайта строку font-family в CSS и готово;

font-family: 'Indie Flower', cursive;
<span style="font-family: 'Indie Flower', cursive;">Some text</span>
Some text

II способ (через @font-face)

1. Скачиваем шрифт с Google Fonts, Webfont, Fontriver

2. Подключаем с помощью к CSS-файлу@font-face (CSS-код может быть вместе со скаченным шрифтом, нужно вставить на самый верх CSS-кода)

@font-face {
    font-family: 'Anfisa Grotesk';
    src: local('Anfisa Grotesk'),
    url('../fonts/anfisa/anfisagrotesk.woff2') format('woff2'),
    url('../fonts/anfisa/anfisagrotesk.woff') format('woff'),
    url('../fonts/anfisa/anfisagrotesk.ttf') format('truetype');
    font-weight: 400;
    font-style: normal;
}
font-family: 'Anfisa Grotesk';
<span style="font-family: 'Anfisa Grotesk';">Some text</span>
 Some text

Для охвата всех вариантов полужирного, курсивного и обычного текста вам нужны четыре директивы (normal), (normal, bold), (normal, italic), (bold, italic) @font-face: с различными font-style: и font-weight:

Иконочный шрифт

1. Создаём свой иконочный шрифт на Glyphter и скачиваем его;
2. Копируем CSS-код в созданном сервисом css-файле и вставляем его.
3. font-family: "Glyphter";

Печатаем в коде: A B C D E F
Получаем в браузере: A B C D E F

Будет грузится следущий через запятую шрифт, если нет первого

font-family: "Times New Roman", Times, serif;

Форматы шрифтов

  • EOT - только в IE;
  • TTF(True Type) и OTF(Open Type) - эти типы шрифтов имеют широкую поддержку в большинстве веб-браузеров;
  • WOFF - Web Open Font Format представляют собой самый новый формат шрифта, разработанный специально для веб-технологий;
  • SVG - Scalable Vector Graphic формат (масштабируемой векторной графики) сам по себе не является форматом шрифта.

Чтобы гарантировать предоставление шрифтов как можно большему количеству браузеров, нужно как минимум использовать шрифты форматов EOT, WOFF и TrueType.
