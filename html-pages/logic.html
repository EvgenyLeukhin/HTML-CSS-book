Hotkeys

  • Space - play/stop
  • Enter - перемотать на начало
  • Play - play from last locate position
  • Z - zoom
  • Backspace - delete track or section
  • up, down - change tracks
  • command + up, down, left, right - scale tracks
  • R - start record
  • E - edit
  • H - hide
  • Q - quantize (выравнивание нот в режиме редактировании миди - нужно выбрать длительность)
  • X - equalizer
  • Alt + move - Копирование (либо на +)

Hotkeys for screens

  • cmd + 1 - main screen
  • cmd + 2 - mixer screen
  • cmd + 3 - instrument controls
  • cmd + 4 - MIDI edit screen
  • cmd + 5 - Notes edit screen
  • cmd + 6 - Wave graph screen
  • cmd + 7 - History
  • cmd + 8 - Files browser
  • cmd + K - Keyboard

Tools

  • Стрелка (pointer tool) - перемещение трека по линейке
  • Прицел (markquee tool) - нарезка
  • Кисточка (glue tool) - склейка
  • Fade - медленное угасание
  • Pencil - MIDI notes
