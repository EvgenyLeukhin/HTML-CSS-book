Таблицы
<table>
  <caption>Заголовок таблицы</caption>
    <tr>
        <th>Заголовок ячейки №1</th>
        <th>Заголовок ячейки №2</th>
        <th>Заголовок ячейки №3</th>
    </tr>
    <tr>
        <td>Ячейка №1-1</td>
        <td>Ячейка №1-2</td>
        <td>Ячейка №1-3</td>
    </tr>
    <tr>
        <td colspan="2">Ячейка №2-1</td>
        <td rowspan="2">Ячейка №2-3</td>
    </tr>
    <tr>
        <td>Ячейка №3-1</td>
        <td>Ячейка №3-2</td>
    </tr>
</table>
Заголовок таблицы
Заголовок ячейки №1 Заголовок ячейки №2 Заголовок ячейки №3
Ячейка №1-1 Ячейка №1-2 Ячейка №1-3
Ячейка №2-1 Ячейка №2-3
Ячейка №3-1 Ячейка №3-2

    Объединение ячеек:

  • colspan="2" - гор. объединение
  • rowspan="2" - вер. объединение

    • Свойства CSS:

  • border-spacing: 5px; - расстояние между границами
  • border-collapse: separate; - границы разделены(по ум.)
  • border-collapse: collapse; - границы слиплись
  • cellspacing: 5px; - отступы между ячейками
  • border-spacing: 5px 10px; - отступы между ячейками
  • table-layout: fixed; - ???
  • или padding для ячеек
  • caption-side: |top|bottom| - расположение заголовка таблицы
  • <thead> - группировка рядов хедера таблицы
  • <tfoot> - группировка рядов шапки таблицы
  • <tbody> - группировка рядов тела(контента) таблицы
  • <col> - колонка таблицы
  • <colgroup> - группа колонок таблицы

При border-collapse: collapse; не будет работать border-spacing:
