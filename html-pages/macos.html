Установка программ через консоль Homebrew

Homebrew

Нужно установить пакетный менеджер Homebrew. Это аналог apt-get на Linux.

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Git

brew install git

Wget

brew install wget

nodejs

brew install node

yarn

brew install yarn
