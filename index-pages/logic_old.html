Hotkeys

Tools

  • Стрелка (pointer tool) - перемещение трека по линейке
  • Прицел (markquee tool) - нарезка
  • Кисточка (glue tool) - склейка
  • Fade - медленное угасание
  • Pencil - MIDI notes

Create new track

  1. Software instrument - MIDI-keyboard
  2. Audio - from the inputs (MICs or instruments from inputs)
  3. Guitar or bass - preset for guitars
  4. Drummer - preset for drumms
  5. External MIDI - ???

Arrangement

  1. Intro
  2. Verse
  3. Chorus
  4. Bridge
  5. Outro

Drummer

  1. Use arrangement
  2. Use follow to track
  3. Use copy to midi (you can change sounds from copied drum part)

Automatization mode

  1. Click automatization
  2. Change Read to Latch
  3. Click record and change params
  4. Params will be change durring recording

Flex mode for instruments

  1. Click common flex mode
  2. Click track flex mode
  3. Flex time - Rhytmic
  4. Add flex-points
  5. Если флекс курсор посередине - то будет правка между линиями, если сверху то по всему треку

Flex mode for vocals

  1. Click common flex mode
  2. Click track flex mode
  3. Flex Pitch
  4. Edit mode - double click (autotune)

Mixdown

  1. Ctrl + A
  2. File -> Bounce -> Project or Section
  3. PCM (Wave), MP3 -> OK

Определить темп

  1. Audio FX -> Metring -> BTM Counter

Расширенные настройки

  1. Cmd + ,
  2. Advanced
  3. Show advanced tools

Record track from OS

  1. Focusrite Control -> Loopback 1-2 -> Playback 1-2 (Stereo) -> Unmute
  2. Logic Pro -> New audio track -> Input 5-6 -> No input monitoring -> Mute track
  3. Play system audio signal -> You can hear it from System playback on your output device
