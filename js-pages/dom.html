DOM
Document object model Связь JS с html осуществляется через DOM, иерархическую структуру html-документа, где все элементы представлены в виде дерева. Это DOM-дерево браузер автоматически отрисовывает при загрузке html-страницы Через это DOM-дерево страниц взаимодействует JS.
Начальная структура DOM-дерева
1. HTMLHtmlElement
2. HTMLBodyElement 		Работа с DOM - это работа с методами, поэтому все ключевые фразы записываются через точку
document
document
(весь html-документ) 		document.методы
 ВМЕСТО document можно писать НАЗВАНИЕ ПЕРЕМЕННОЙ, в кот. сохранён какой-либо DOM-элемент 		document - корень DOM-дерева страницы в браузере
Так язык JS взаимодействует с веб-страницей через DOM
location
location location.href // выведет текущий адрес находится в window
window
window
(ОКНО, внутри кот. находится DOM-дерево), всё нохадится в нём, даже document 		window.методы По умолчанию стоит у методов alert, prompt, confirm
Примеры
Задание цвета для body //задали цвет:
document.body.style.backgroundColor = 'red';
//сбросили цвет:
document.body.style.backgroundColor = '';
Теги html и body <HTML> = document.documentElement;
<BODY> = document.body;
Выбор элементов в DOM $('.class').get(0); Выбор первого элемента с классом class
data-атрибуты
Можно добавлять в любые теги, должен быть префикс data- <div data-filter="first'></div>
document.querySellector('div').dataset.filter; // 'first' 		С помощью них можно хранить любую информацию
