Стилизация

I способ (old-school)

При билде вся стилизация в одном файле bundle.css в тег link 		<link src="/bunble.css"> Работу по импорту css-файлов в App.js осуществляет Webpack

II способ (most useage)

На каждый импорт будет создан отдельный тег style 		// импортируем стили в компонент // глобальные стили в index.js import './SomeComponent/SomeComponent.css'; Работу по импорту css-файлов в App.js осуществляет Webpack

III способ (JSS - hardcore)

Инлайн-стилитизация, CSS in JS 		render() {   // после рендера создаём объект style   const style = {
     color: 'white',
     fontSize: '20px',
     border: '1px solid #ccc',
     backgroundColor: 'blue'
  };

  ...
}
// а замем нужно присвоить этот объект атрибуту style <button style={style}> 		css-свойства должны записываться в js-формате (в camelCase).
Можно подключать библиотеки для этого способа (такие как radium, glamor, styled-components)
