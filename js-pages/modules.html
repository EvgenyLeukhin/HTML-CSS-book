Classes

С помощью классов можно существенно сократь дублирование кода, за счёт расширения дочернего класса другим родительским классом. Повторное использование кода

Классы принято именовать с большой буквы
class Parentclass {
  constructor() { // для свойств
    this.prop1 = 'prop1'
    this.prop2 = 'prop2'
    this.prop3 = 'prop3'
  }

  someFunc() { ... }
};

// создаём новый класс и наследуем св-ва род. класса
class Childclass extends Parentclass {
  constructor() {
    super();
    this.prop1 = 'prop1+'
  }
}
// теперь этот класс унаследовал все св-ва род. класса
// и можно изменять св-ва род. класса и функции
const some = new Childclass();
some.someFunc();
some.prop2 = 'another';
Через BABEL constructor(), super() и this можно не записывать (ES7)

Очень похожи на объекты, только записываем = вместо : для присваивания значенией и не нужно запятых между свойствами.

function-конструктор
Конструкторы принято называть с большой буквы // создание конструктора
var Person = function(name) {
  this.name = name;
  return console.log(this.name);
};

// вызов функции-конструктора (new)
var person = new Person('Some name'); 		КОНСТРУКТОР - это функция, возвращающая объект через ключевое слово this. (Функция, создающая объект с параметрами)

Принимает свойство name и присваивает ему значение.

Можно вызвать этот конструктор через ключевое слово new.
class-конструктор
Вместо var используем ключевое слово class 
// создаётся класс с большой буквы (класс это объект)
class Person {
  // создаётся функция-конструктор этого класса,
  // присваивающая свойству name этого объекта
  // значение 'MAX'
  constructor() { this.name = 'MAX'; }

  // создаётся функция класса, выводящяя значение
  // свойства name этого класса
  printMyName() {
    console.log(this.name);
  }
}

// используем конструктор класса и создаём новый объект
// сохраняем его в константу
const person = new Person();

// теперь person это тоже объект со свойством name = 'MAX'
person.printMyName();  // 'MAX'

// но его можно изменить
person.name = 'Eugin';
person.printMyName();  // 'Eugin'
В ES6 конструкторы создаются с помощью классов и стандартной функции constructor
class extends
Наследование класса class Person {
  constructor() { this.name = 'MAX'; }
  printMyName() { console.log(this.name); }
}

// создаём новый класс, который наследует свойства другого класса
class Gender extends Person {
  constructor() {
    // указываем функцию super для наследавания свойств
    super();
    this.gender = 'male';
  }
  printMyGender() { console.log(this.gender); }
}

let some = new Gender();
console.log(some.name);
console.log(some.gender);
