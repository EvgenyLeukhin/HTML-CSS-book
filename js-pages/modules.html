Структура проекта для фронтэнда
1. node.js + npm Устанавливаем node.js+ npm для вашей ОС (делается 1 раз вручную)
 https://nodejs.org/en/download/
2. GitHub 1. Устанавливаем себе консоль Git bash (делается 1 раз вручную) https://git-scm.com/download
2. Регистрируемся и авторизуемся (Sign in) на github.com
3. Создаём репозиторий (New repository + Initialize this repository with a Readme.md)
4. Клонируем репозиторий себе на жёсткий диск (Clone or download), копируем адрес, открываем консоль, встаём в нужную директорию, пишем команду: git clone + адрес репозитория
5. Создаём .gitignore (файлы, кот. НЕ будут отправляться)
dist/
node_modules/
Thumbs.db
*.psd
*.pdf
6. Редактируем Readme.md (информация о проекте)
# IT-book (Russian language)
<hr>
My IT-book
<br>
Author: Evgeny Leukhin
<br>
Mail: EvgenyLeukhin@ya.ru
<br>
VK: <https://vk.com/leukhin_ei>
<br>
GutHub hosting: <https://evgenyleukhin.github.io/IT-book/>
3. Создаём директории проекта 
dist/                  директория собранного проекта
src/		       рабочая директория
	../_design     файлы дизайна
	../coffee      файлы coffee
	../css         файлы css
	../docs        документы, прочие файлы
	../fonts       шрифты
	../img         изображения для дизайна
	../js          js-скрипты
	../js/libs     js-библиотеки
	../less        файлы less
	../sass        файлы sass
	../pict        изображения для контента
	../index.html  основной html-файл
4. Инициализация проекта Тут три варианта:

1. Создавать настройки с нуля, прописать в директории с проектом команду npm init и создать все настройки вручную

2. Взять уже готовый файл package.json для типовых проектов c готовым набором devDependencies, и в нём отредактировать при необходимости name, description, repository, keywords

3. Создать файл package.json вручную; 		Будет создан package.json с зависимомтями
5. bower 1. Устанавливаем глобально node-пакет bower для загрузки библиотек npm i -g bower
(делается 1 раз)
2. Создаём в главной директории с проектом .bowerrc, в котором указываем директорию назначения для установки библиотек:
{
  "directory": "src/js/libs/"
}
3. В консоли устанавливаем все нужные для проекта библиотеки
bower i bootstrap jquery react angular
6. gulp и gulpfile.js gulpfile.js Тут два варианта:

1. Создать gulpfile.js вручную, прописав в нём все необходимые настройки для данного проекта

2. Взять готовый gulpfile.js, и использовать уже типовой набор настроек
Установка gulp и gulp плагинов Тут тоже два варианта:

1. Установить gulp и все плагины вручную, прописав команду в консоли:
npm i --save-dev gulp gulp-less gulp-watch gulp-concat и т. д.

2. Установить всё из готового package.json, прописав в консоли команду: npm i
