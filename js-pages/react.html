React-элемент
Создание React-элемента // <h1>Some text</h1> var h1 = React.createElement(
  'h1',
   null,
  'Some text'
);
// <h1 class="some-class">Some text</h1> var h1 = React.createElement(
  'h1',
   {className: 'some-class'},
  'Some text'
); 		Принимает три параметра:
1 - какой элемент создавать;
2 - параметры элемента
3 - содержание элемента

Могут быть также различные переменные в третьем параметре (принимает сколь угодно значений)
jsx
Этот формат симбиоз js и html. В нём можно записывать html-код как и в html-файлах без кавычек. Транспилируется с помощью транспилятора babel
Параметры
Параметры объекта второго
аргумента функции React.createElement()		 className - class
title - title
style - style
id - id
href - href
Нестандартные параметры отрендерены не будут, но React про них будет помнить

id тега в html-разметке
React-компонент
Компоненты дают возможность многократного использования // создаём компонент ReactComp class ReactComp extends React.Component {
  return React.createElement('div', null, h1,h1,h1);
}
// теперь можно его отрендерить ReactDOM.render(
  React.createElement(HelloReact, null),
  document.getElementById('content')
); // или так div>div*3>h1+h1+h1

Компоненты без параметров

ReactDOM.render(

Компоненты с параметрами

...
  React.createElement(HelloReact, {
    "title": "first",
    "className": "some-class1",
    "id": "some-id1"
  }),
  React.createElement(HelloReact, {
    "title": "second",
    "className": "some-class2",
    "id": "some-id2"
  }),
  React.createElement(HelloReact, {
    "title": "third",
    "className": "some-class3",
    "id": "some-id3"
  }),
...

Можно выводить эти параметры, поменяв компонент

class ReactComp extends React.Component {
  return React.createElement('div', null,
  'text ' + this.props.title);
}

И присвоить все параметры

class ReactComp extends React.Component {
  return React.createElement('div', this.props,
  'text ' + this.props.title);
}

Шаблон React-компонента

// Создаём функцию компонента с параметрами 
function Person(props) {
  return (
    <div className="person">
      <h1>{props.name}</h1>
      <p>Your Age: {props.age}</p>
    </div>
  );
}

// сохраняем шаблон компонента в переменную
var app = (
  <div>
     <Person name="Max" age="28" />
      <Person name="Manu" age="29" />
  </div>
);

// рендерим этот шаблон в #app
ReactDOM.render(app, document.querySelector('#app'));
 Результат будет такой же как в коде выше
