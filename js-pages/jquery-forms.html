Выборки элементов формы
:input - все инпуты
:text - все инпуты c type="text"
:password - все инпуты c type=""
:radio - все инпуты c type=""
:checkbox - все инпуты c type=""
:submit - все инпуты c type=""
:reset - все инпуты c type=""
:file - все инпуты c type=""
:hidden - все инпуты c type=""
:image - все кнопки-рисунки
:button - все button
// все текстовые поля в форме
$('#form :text')
События-состояния форм и методы
:selected - для select
:checked - для checkbox
.submit() - отправка формы
.focus() - поле в фокусе
.blur() - поле без фокуса
.change() - изменение значения
.val() - значение в поле формы
.prop() - свойство формы

Добавление атрибута :checked при клике

$('conteiner-class').click(function() {
  $('input', this).prop('checked', true);
});
