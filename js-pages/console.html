node.js в консоли
1. Установка node.js node - переключение в режим Node.js-консоли
.exit или Ctrl + C - выход
node script.js - выполниться script.js в консоли 		Консоль с интерпретатором JavaScript-кода
2. Установка npm После установки node.js, npm уже установлен, но для каждого проекта нужен свой набор моделей. Встаём в папку с новым проектом и набираем:
npm install
Структура node-модулей (после установки npm):
node-modules/ - модули
src/ - файлы проекта
test/
index.js/ - указывает на путь js-файлов проекта, копирует весь js-код в себя
Makefile
npm-debug.log
package.json - файл настроек
Readme.md - информация о проекте

npm на практике

npm run \название скрипта\- запуск скриптов в файле package
npm run test - запуск теста
npm run eslint index.js - запуск ESLint
 		Пакетный менеджер для node.js
Установщик различных модулей, приложений, библиотек
package.json - главный файл с настойками (с ключами)
3. Установка ESLint

Глобальная установка:

npm install -g eslint
npm install -g --save-dev eslint-config-airbnb
npm install -g --save-dev eslint-plugin-import
Заменить на //.eslintrc
"extends": "airbnb/base"

ESLint на практике

eslint file or folder
eslint file or folder --fix //Исправит пробелы 		Теоритически при команде npm install этот пакет уже должен поставить в папку node-modules/, но можно его поставить как локально в проект, так и глобально
Файл настроек .eslintrc
4. Установка browser-sync

Установка:


npm install --save-dev browser-sync

Подключить браузер:

Встать в консоли в нужную директорию и набрать команду:

browser-sync start --server --directory --files "**/*" 		И сайт в режиме разработки будет запускаться с локального сервера по адресу: http://localhost:3000/
5. Установка bower

Установщик библиотек

Установка:


npm i -g bower
Далее создаём файл в директории с проектом .bowerrc, в котором указываем директорию назначения для установки библиотек: 

{
	"directory": "app/js/libs/"
}
Теперь можно устанавливать библиотеки командой:
bower i jquery magnific-popup
