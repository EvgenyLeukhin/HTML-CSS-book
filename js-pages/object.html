Объекты
Вызов объекта

Состоит из пар: имя-значение или ключ-значение

Синтаксис: {
  имя: значение,
  имя: значение,
  имя: значение
}; 		var myObject = {
  model: 'Some string',
  name: 'Some name',
  cost: 5
};
myObject.cost = 10;
//изменение значений переменных в объектах 		Объект вызывается с помощью фигурных скобок {...} - поле объекта, в которые вставляются свойства данного объекта. Свойства в объектах перечисляются через , присваиваются через :
Вызов свойств объекта
Также можно их изменять 		var myObj = {
  name: "John",
  age: 30,
  gender: "male"
};
Первый способ: console.log(myObj.name);
Второй способ: console.log(myObj['name']); 		Через document notation
Вложенные (многомерные) объекты var myObject = {
  model: 'Some string',
  name: {
    firstName: 'John',
    lastName: 'Smith',
    nickName: 'Robot'
  },
  cost: 5
};

Доступ к вложенному объекту:

console.log(myObject.name.firstName);
Изменение свойств из объекта myObject.name = "New name";
Удаление свойств из объекта delete myObject.age;
Проверка наличия свойства в объекте "name" in myObject; //true
Запись свойств объекта в кавычках 
var demoObject = {
  'son1': 'Абырвалг',
  'son2': 'Телеграф',
  'son3': 'Телефон'
};

for (var i = 1; i <= 3 ; i++) {
  var sonName = demoObject['son' + i];
  console.log('Сын ' + i + ': ' + sonName);
}
Можно прочитать название свойств в цикле
