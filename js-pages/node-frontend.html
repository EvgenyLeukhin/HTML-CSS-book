Frontend

ESLint

.eslintrc.js 		npm i -g eslint
npm i --save-dev eslint

Если нужно создать конфиг (.eslintrc.js) с нуля:
----------------------------------------------------
eslint --init

Или взять готовый(.eslintrc.js):
---------------------------------------------------- 
module.exports = {
  "env": { "browser": true },
  "extends": "eslint:recommended",
  "rules": {
    "indent":          [ "error", 2        ],
    "linebreak-style": [ "error", "unix"   ],
    "quotes":          [ "error", "single" ],
    "semi":            [ "error", "always" ]
  }
};

Запуск в консоли:
----------------------------------------------------
eslint some-file.js
eslint some-file.js --fix
Чтобы ESlint работал в горячем режиме в редакторе кода, для редактора нужно установить плагин ESLint.

B Sublime нужно установить два плагина (SublimeLinter, SublimeLinter-eslint).

Если установлен плагин для ESLint, то в конфиге пишем "extends":"plugin-name"

browser-sync

 npm i -g browser-sync
npm i --save-dev browser-sync

Запуск в консоли в директории с проектом:
----------------------------------------------------
browser-sync start --server --directory --files "**/*" 		Будет выполнен запуск файла index.html в браузере, который лежит в текущей директории. И браузер будет перезагружаться при изменении всех файлов в директории
