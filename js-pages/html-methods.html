Мой Справочник | JavaScript | Методы работы с DOM

Поиск элементов

.querySelector()

 document.querySelector('.some-class'); Выбрать элементы по любому селектору как в CSS

.querySelectorAll()

 document.querySelectorAll('.some-class'); Выбрать все элементы по любому селектору как в CSS

.getElementById()

 document.getElementById('some-id'); Выбрать элемент по id

.getElementName()

 document.getElementByName('some-id'); Выбрать элемент по атрибуту name

.getElementsByClassName()

 document.getElementsByClassName('some-class'); Выбрать элемент по классу

.getElementsByTagName()

 document.getElementsByTagName('p'); // all elements document.getElementsByTagName('*'); // first paragraph document.getElementsByTagName('p')[0]; Выбрать элементы по названию тега
classList

.classList.add()

 someElement.classList.add('some-class'); Добавить класс к элементу someElement

.classList.remove()

 someElement.classList.remove('some-class'); Удалить класс у элемента someElement

.classList.toggle()

 someElement.classList.toggle('some-class'); Переключает класс у someElement на some-class

.classList.contains()

 someElement.classList.contains('some-class'); Проверка на наличие класса
Attributes

.setAttribute()

 someElement.setAttribute('style', 'color: red;'); Добавить атрибут или атрибут со значением

.hasAttribute()

 someElement.hasAttribute('style'); Проверка на наличие атрибута

.getAttribute()

 someElement.getAttribute('style'); Получить значение атрибута

.removeAttribute()

 someElement.removeAttribute('style'); Удалить атрибут

.className()

 someElement.className('some-class'); Добавить класс

.title()

 someElement.title('title text'); Атрибут title

.style()

 someElement.style.color = 'red';
someElement.style.backgroundColor = 'red'; 		Атрибут style. У каждого css-свойства есть свой метод в объекте style

.hidden

 someElement.hidden = true;
 Атрибут hidden. Не функция, скрывает, но не удаляет из DOM
forms

.forms

 docuement.forms;
// форма с id var someForm = docuement.forms['some-id'] // поля этой формы c id someForm['some-input-1'];
someForm['some-input-2'];
DOM-коллекция всех форм

.validity

 someElement.validity.ValidityState;
 Объект и подобъект, кот. хранят в себе валидацию

.value

 someElement.value = 2; Присваивание значений полям

.min
.max

 someElement.min = 0;
someElement.max = 10; 		Макс. и минимальные значения полей

.selected

 someElement.selected = true;

.checked

 someElement.checked = true;
dataset

.dataset()

 <div data-filter="active"></div>
// значение этого data-атрибута someElement.dataset.filter; 		Можно получить или изменить значение любых data-атрибутов
OtherFunctions

.get()

 someElement.get(0); Выбор элемента по порядку в коде. Первый элемент в коде

innerHTML()

 someElement.innerHTML;
// change html-code someElement.innerHTML = '<p>Something'<p>;
 Можно сохранять или изменять, вложенный внутрь элемента html-код

.createElement()

 document.createElement('LI');
 Создание нового элемента

.appendChild()

 el.appendChild(someVar);
 Добавить элемент в конец

.prepend()

 el.prepend(someVar);
 Добавить элемент в начало

.children()

.remove()

 someElement.remove();
 Функция, полностью удаляющаяя элемент из DOM
localStorage

.setItem()

 localStorage.setItem('name', 'John Smith'); Сохранить значение в хранилище.

name - переменная (ключ), внутри которой будет храниться 'John Smith'

.getItem()

 localStorage.setItem('name'); Получить значение ключа в хранилище.

Если у переменной name есть сохранённое значение в localStorage, то этот метод вернёт это значение

.removeItem()

 localStorage.removeItem('name'); Удалить значение ключа из хранилища.

Если у переменной name есть сохранённое значение в localStorage, то этот метод УДАЛИТ это значение

.clear()

 localStorage.clear(); Очистить всё хранилище.

Метод очищает все сохранённые значения в хранилище
Cookies

Сохранять значения в сookies

 document.cookie = 'myFirstCookie=Something'; // show all cookies alert(document.cookie); // создаём шаблон даты для cookie (3 дня) var dateToExpire = +Date.now() + 3*24*60*60*1000; // переводим её в читаемый вид для cookie var convertFormatDate = new Date(dateToExpire).toUTCString(); // сохранение значений полей document.cookie = 'inputVar=' + inputVar.value + ';expires=' + convertFormatDate; Сохраняем в ключ myFirstCookie значение Somethind. Эти сохранённые значения храняться на сервере тоже. Не более 4KB. Для больших объёмов нужно использовать localStorage

Есть библиотека cookies.js для упрощеия работы с куками. Подключантся в <head>

Чтобы сохранить cookie нужно при событии отправки (onsubmit),отправку остановить e.preventDefault(), затем сохранить введённые значения в куки, а потом отправить форму .submit()
