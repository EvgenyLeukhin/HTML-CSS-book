querySelector('селектор');
Получение элемента по СЕЛЕКТОРУ
querySelector() - поиск элемента по селектору как написано в css-коде 		document.querySelector('.user-block');
 document - ищем по всему документу
. - все методы у объектов пишутся через точку
querySelector - методом селекторов
'.user-block' - селектор с таким именем
var link = document.querySelector('.user-block'); 		Найдёт первый попавшийся элемент в коде. Все методы можно сохранять в переменные
querySelectorAll('селектор');
Получение элементов по СЕЛЕКТОРУ
querySelector() найдёт только один элемент, если их много нужен данный метод 		document.querySelectorAll('.main-menu li');
 document - ищем по всему документу
querySelector - методом селекторов
'.user-block' - селектор с таким именем
var link = document.querySelector('.main-menu li'); 		Найдёт первый попавшийся элемент в коде. Все методы можно сохранять в переменные
classList
Набор методов для управления классами

Добавление нового класса:

element.classList.add("new-class");
element - элемент, кот. будет добавлен новый класс;

Удаление класса:

element.classList.remove("some-class");
element - элемент, у кот. будет удален класс;

Переключение класса:

element.classList.toggle("some-class");
element - элемент, у кот. будет переключён класс;

Сообщает есть ли класс у элемента:

element.classList.contains("some-class");
element - элемент, у кот. будет переключён класс;
className
Присвоить класс document.querySelector('h1').className = 'some-class';
localStorage
Локальное хранилище 
Сохранить значение в хранилище:
localStorage.setItem('name', 'John Smith');
name - переменная (ключ), внутри которой будет храниться 'John Smith' 
Получить значение ключа в хранилище:
localStorage.getItem('name');
Если у переменной name есть сохранённое значение в localStorage, то этот метод вернёт это значение 
Удалить значение ключа в хранилище:
localStorage.removeItem('name');
Если у переменной name есть сохранённое значение в localStorage, то этот метод УДАЛИТ это значение 
Полностью очистить хранилище:
localStorage.clear();
Метод очищает все сохранённые значения в хранилище
