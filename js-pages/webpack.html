Webpack - Курс Udemy

Описание

 Webpack - это bundler (упаковщик).
Его работа включает 4 этапа:
  1. Entry point (Точка входа. App.js или index.js)

  2. Loaders - загрузчики (они загружают и преобразуют файлы). Их работа состоит в компиляции файлов (JSX -> js, less/scss -> css)

  3. Plugins - плагины (они выполняют разлючные функци, например, минификацию)

  4. Output - выходная точка (build/bundler.js)

Babel для Webpack

 // плагин для запуска rollup npm i --save-dev --save-exact rollup-plugin-plugin-babel // пресет env - для каких браузеров (по ум* 2 последние версии браузеров) npm i --save-dev --save-exact babel-preset-env // плагин для недублирования функций npm i --save-dev --save-exact babel-plugin-external-helpers // полифил для поддержки нового ES (--save) npm i --save --save-exact babel-polyfill // полифил для поддержки старого ES (--save) npm i --save --save-exact whatwg-fetch

webpack.config.js

 

// подключение модулей
const path = require('path');
const webpack = require('webpack');


// настройка 
module.exports = {
  entry: './index.js',
  output: { path: __dirname, filename: 'webpack_bundle.js'},
  module: {
    loaders: [
      {
        test: /.jsx?$/,
        loader: 'babel-loader',
        exclude: /node_modules/,
        query: {
          presets: ['es2015', 'react']
        }
      }

  }
