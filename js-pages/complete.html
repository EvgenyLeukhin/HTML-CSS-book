Мой Справочник | JavaScript | Вывод данных

alert
Вызов окна оповещения браузера alert();
alert('Hello, World!'); 		В () можно писать что угодно (текст, значения, операции и т.д.)
Полная запись window.alert(), window означает окно браузера, стоит по ум*, поэтому можно опустить
prompt
Вызов окна оповещения браузера с вводным полем prompt();
prompt('Ваше имя: ');
name = prompt('Ваше имя: '); 		Можно использовать для присваивания значений переменных
Консоль
Вывести результат в консоль инспектора браузера console.log();
console.warn(); - предупреждение,
console.info(); - информация,
console.error(); - ошибка 		Вызов консоли: Ctrl + Shirt + J Результат будет виден в инспекторе браузера F12 - Консоль - F5 (Обновить страницу).
document.write
Добавление информации на страницу document.write();
document.write('<p>Hello, World!</p>'); 		Полностью затирает html-код на странице и добавляет новый
confirm
Всплывает окно с двумя кнопками OK(true) и ОТМЕНА(fasle) if(confirm('Are you sure?')) {
  alert('Hello!');
} else {
  alert('Goodbye!');
} 		Можно применять вместе с условными операторами для гибкости ответа программы
