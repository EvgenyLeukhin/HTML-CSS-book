Присваивание (=)
Оператор для присваивания значений переменным a = b = c = 2 + 2; Такой код будет выполняться справа-налево. Сначала выполнится 2 + 2, а уже затем будут выполняться присваивания c, b, a
Перечисление (,)
Запятая тоже оператор. Оператор перечисления a = 1,
b = 2,
c = 3;
Доступ (.)
Точка - оператор доступа до свойств объекта a.b; Свойство b объекта a
Сложение (+)
Сложение переменных (конкатенация)

Цифровое сложение

var a = 1;
var b = 2;
var c = a + b;
console.log(c); //3

Строковое сложение

var a = 'John ';
var b = 'Smith';
var c = a + '' + b;
console.log(c); //John Smith

Смешанное сложение

var a = 1;
var b = 'Smith';
var c = a + b;
console.log(c); //1Smith 		Если при сложении различных переменных хотя бы одно будет 'строкой', то интерпретатор присваивает остальным тоже строчный тип. У них приоритет.
Так работает ДИНАМИЧЕСКОЕ ПРИВЕДЕНИЕ ТИПОВ
Вычитание (-)
Вычитание переменных var a = 1;
var b = 2;
var c = a - b;
console.log(c); //-1
Умножение (*)
Умножение переменных var a = 1;
var b = 2;
var c = a*b;
console.log(c); //2 		Если стоит бинарный оператор умножения, то интерпретатор будет пытаться преобразовать операнды в числовой тип
console.log('2'*'3'); //6
Деление (/)
Деление переменных var a = 1;
var b = 2;
var c = a/b;
console.log(c); //0.5 		Если стоит бинарный оператор деления, то интерпретатор будет пытаться преобразовать операнды в числовой тип
Остаток от деления (%)
Возвращает целвй остаток от деления чисел 10 % 3; // 1
10 % 4; // 2
10 % 5; // 0
Приоритет
Приоритет у умножения и деления, но если поставить (), то приоритет изменится e = a + b * c / d;
f = (a + b)* c / d;
 		То, что стоит в скобках, то приоритетней, как в математике.

При сложении числа и строки интерпретатор будет преобразовывать число в строку.

При умножении или делении будет наоборот
Краткая запись
Краткая запись операторов ++ Увеличивает на 1 (ИНКРИМЕНТ)
++a Префиксный инкримент, увеличивает сразу на 1
a++ Посткрисный инкримент, увеличивает на 1, но возвращает старое значение
-- Уменьшает на 1 (ДИКРИМЕНТ)
a = (10%3); //1 Остаток от деления
a += 10; Аналог: a = a + 10;
a -= 10; Аналог: a = a - 10;
a *= 10; Аналог: a = a * 10;
a /= 10; Аналог: a = a / 10;

Логические операторы
Чаще всего применяются для условных операторов if >   - больше
>=  - больше или равно
<   - меньше
<=  - меньше или равно
==  - нестрогое равно
=== - строгое равно
!=  - нестрогое неравно
!== - строгое неравно
&& - И
|| - ИЛИ
a && b - если a true, то b не проверяется
a || b - если a false, то b не проверяется
'1' || '2' || '3' //1
'1' && '2' && '3' //3
 		Операторы += и другие, добавляют к переменной то, что располагается справа от знака =
Операторы > Больше и < Меньше так же работают со СТРОКОВЫМ ТИПОМ ДАННЫХ, больше будет, то слово, чья буква дальше по алфавиту, причём строчные буквы стоят дальше заглавных, поэтому больше ('a' > 'A')
Логический оператор || проверяет на истинность левый операнд, если он тру, то остальные не проверяет, а сразу возвращает значение этого правдивого операнда, если левый ложь, тогда проверяет правый
Логический оператор && также начинает проверять левый операнд, если он ложный, то другое не проверяется, а сразу возвращает ложь первого
ИЛИ вспотыкается на правде, И - на лжи
Операторы вида ++ применяются ТОЛЬКО К ПЕРЕМЕННЫМ, нельзя записывать ++5
... (Оператор распространения ES6)
Оператор распространения let oldArray = [1, 2, 4];
let newArray = [...oldArray, 8]; // [1, 2, 4, 8]
// тоже самое для объектов
let oldObj = {pro1: val1, pro2: val2, pro3: val3};
let newObj = {...oldObj, pro4: val4};
// также можно склеивать аргументы функции в массив и применять к ним методы для массивов
function sortArg(...args) {
  return sortArg.sort();
} 		Вставить куда-либо элементы массива
Побитовые операторы
Перемещение 1 в двоичной системе исчисления

Побитовое И

1 & 2;

Побитовое ИЛИ

1 | 2;
// 0 меняются на 1 в бинарном коде (001 | 010 = 011)

Побитовое НЕ

1 ~ 2;

Побитовые сдвиги вправо-влево

Вправо: деление на 2; влево: умножение на 2 (1 ед.)
a = 32 >> 1; //16
a = 64 << 1; //128 		Побитовые операции работают быстрее, чем остальные 
a = 0;   // 00000000000000000000000000000000
a = 1;   // 00000000000000000000000000000001
a = 2;   // 00000000000000000000000000000010
a = 3;   // 00000000000000000000000000000011
a = 255; // 00000000000000000000000011111111

Описание

Элемент языка

Прибавляет значение выражения к значению переменной и присваивает результат переменной.

Оператор присваивания сложения (+=)

Выполняет сложение двух чисел или объединяет две строки.

Оператор сложения (+)

Присваивает значение переменной.

Оператор присваивания (=)

Осуществляет операцию побитового И для значения переменной и значения выражения, а затем присваивает результат переменной.

Оператор присваивания побитового И (&=)

Выполняет побитовую операцию И для двух выражений.

Оператор побитового И (&)

Сдвигает биты выражения влево.

Оператор побитового сдвига влево (<<)

Выполняет операцию побитового НЕ (отрицания) для выражения.

Оператор побитового НЕ (~)

Выполняет операцию побитового ИЛИ для значения переменной и значения выражения, а затем присваивает результат переменной.

Оператор присваивания побитового ИЛИ (|=)

Выполняет операцию побитового ИЛИ для двух выражений.

Оператор побитового ИЛИ (|)

Сдвигает биты выражения вправо, сохраняя знак.

Оператор побитового сдвига вправо (>>)

Выполняет операцию побитового исключающего ИЛИ для значения переменной и значения выражения, а затем присваивает результат переменной.

Оператор присваивания побитового исключающего ИЛИ (^=)

Выполняет операцию побитового исключающего ИЛИ для двух выражений.

Оператор побитового исключающего ИЛИ (^)

Приводит к последовательному выполнению двух выражений.

Оператор "запятая" (,)

Возвращает логическое значение, показывающее результат сравнения.

Операторы сравнения

Список составных операторов присваивания.

Составные операторы присваивания

Выполняет одно из двух выражений в зависимости от условия.

Условный (троичный) оператор (?:)

Уменьшает переменную на единицу.

Оператор уменьшения (--)

Удаляет свойство из объекта или элемент из массива.

Оператор delete

Делит значение переменной на значение выражения и присваивает результат переменной.

Оператор присваивания деления (/=)

Выполняет деление двух чисел и возвращает числовой результат.

Оператор деления (/)

Проверяет существование свойства в объекте.

Оператор in

Сравнивает два выражения и определяет, равны ли они.

Оператор равенства (==)

Сравнивает два выражения и определяет, верно ли, что одно значение больше другого.

Оператор "больше чем" (>)

Сравнивает два выражения и определяет, верно ли, что одно значение больше другого или равно ему.

Оператор "больше чем или равно" (>=)

Сравнивает два выражения и определяет, имеют ли они равные значения и один и тот же тип данных.

Оператор тождества (===)

Увеличивает переменную на единицу.

Оператор инкремента (++)

Сравнивает два выражения и определяет, верно ли, что они не равны.

Оператор неравенства (!=)

Возвращает логическое значение, указывающее, является ли объект экземпляром определенного класса.

Оператор instanceof

Сдвигает значение переменной влево на количество битов, указанное в значении выражения, и присваивает результат переменной.

Оператор присваивания сдвига влево (<<=)

Сравнивает два выражения и определяет, верно ли, что одно значение меньше другого.

Оператор "меньше чем" (<)

Сравнивает два выражения и определяет, верно ли, что одно значение меньше другого или равно ему.

Оператор "меньше чем или равно" (<=)

Выполняет логическое объединение двух выражений.

Оператор логического И (&&)

Выполняет логическое отрицание выражения.

Оператор логического НЕ (!)

Выполняет логическое разделение двух выражений.

Оператор логического ИЛИ (||)

Делит значение переменной на значение выражения и присваивает остаток переменной.

Оператор присваивания модуля (%=)

Выполняет деление двух чисел и возвращает остаток.

Оператор модуля (%)

Умножает значение переменной на значение выражения и присваивает результат переменной.

Оператор присваивания умножения (*=)

Выполняет умножение двух чисел.

Оператор умножения (*)

Создает новый объект.

Оператор new

Сравнивает два выражения и определяет, верно ли, что они не имеют равные значения или один и тот же тип данных.

Оператор нетождества (!==)

Сдвигает значение переменной вправо на количество битов, указанное в значении выражения, с сохранением знака и присваивает результат переменной.

Оператор присваивания сдвига вправо (>>=)

Позволяет инициализировать массив из нескольких элементов литерала массива или расширить выражение до нескольких аргументов (в вызовах функций).

Оператор Spread

Вычитает значение выражения из значения переменной и присваивает результат переменной.

Оператор присваивания вычитания (-=)

Выполняет вычитание двух выражений.

Оператор вычитания (-)

Возвращает строку, которая идентифицирует тип данных выражения.

Оператор typeof

Обозначает отрицательное значение числового выражения.

Оператор унарного отрицания (-)

Сдвигает значение переменной вправо на количество битов, указанное в значении выражения, без сохранения знака и присваивает результат переменной.

Оператор присваивания сдвига вправо без знака (>>>=)

Сдвигает биты выражения вправо без учета знака.

Оператор сдвига вправо без знака (>>>)

Запрещает выражению возвращать значение.
