Distructing

Диструктивное присваивание

 const [a, b, c, d] = [1, 2, 3, 4];
console.log(a); // 1
console.log(b); // 2
console.log(c); // 3
console.log(d); // 4 		С одной стороны массив с названиями переменных, с другой стороны массив со значениями
let

let

Нельзя повторно переобъявлять

let a = 5;
a = 15; // Изменять можно
let a = 15; // Ошибка!

Scope

let a = 5;
console.log(a); // 5
{
  let a = 15;
  console.log(a); // 15
} 		В отличие от var, scope у let - это блок, в кот. выполянется объявление.

В ES6 можно его использовать всегда вместо var

В {} ограничивается scope у let, и даже переменные с одинаковыми названиями на разных уровнях ничего не знают друг о друге
const

const

Нельзя повторно переобъявлять и изменять

const a = 5;
a = 15; // Ошибка!
const a = 15; // Ошибка!

Scope

const a = 5;
console.log(a); // 5
{
  const a = 15;
  console.log(a); // 15
} 		При объявление этой константы единожды, ей нельзя будет присвоить другое значение, иначе будет ошибка.

Scope у const действует также как и у let. На разных уровнях константы ничего не знают друг о друге
var (ES5)
Вызов переменных var a;
var name;
var answer;
//или в одну строку через запятую
var a, name, answer;
var CONST; // в JS принято называть константы большими буквами 		Чтобы вызвать переменную, нужно записать её имя БЕЗ КАВЫЧЕК!

Переменные бывают ГЛОБАЛЬНЫЕ и ЛОКАЛЬНЫЕ
Присваивание (=) a = 5;
name = 'Jack';
answer = true;
var a = (5, 'Text'); //примет значение 'Text', то что правее 		Присваивание осуществляется посредствам знака = Переменные могут хранить в себе любой тип данных. Переменную можно сравнить с коробкой, в которую можно ложить или вынимать что-нибудь.
После присваивания название переменной хранит в себе значение.
Вызов с присваиванием var a = 5;
var name = 'Jack';
var answer = true; 		Можно вызывать переменные и сразу присваивать им значения
Присваивание через prompt var name = prompt('Ваше имя: ');
var a = prompt('А = ');
a = parseInt(a);
//или в одну строку:
var a = parseInt(prompt('А = ')); 		По умолчанию, если присваиваете значение переменной через prompt, то значение будет 'строкой'.
Если хотите присвоить число, то придётся переконвертировать строку в число, для этого нужна инструкция parseInt
Имена переменных
Название переменной должно начинаться с буквы, _ или $

Правильная запись:

var score;
var score2;
var scoreScore;
var Score;
var _score;
var $score;

Неправильная запись:

var 1score;
var %Score;
var №score;
var &score;
JS чувствителен к регистру var score;
var Score;
var scOre; 		score, Score, scOre различные переменные
Нельзя называть переменную названием оператора JS или ключевым словом var var;
var alert;
var if; 		Можно использовать символ _ перед названием переменной, чтобы не было абсолютного совпадения
