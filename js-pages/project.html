Структура проекта
1. Привязка к GitHub-репозиторию 1. Создаём репозиторий на GitHub
2. Клонируем его себе на хард-диск git clone + адрес на github
3. Создаём Readme.md, информационный файл о проекте
4. Создаём .gitignore npm init 		Будет создан package.json
2. Инициализация проекта npm init Будет создан package.json с зависимомтями
3. Создаём директории проекта 
src/            директория рабочих файлов (или app/)
dist/           директория готовых файлов проекта
node-modules/   директория node-приложений
.bowerrc        файл настроек библиотек  
.editorconfig   файл настроек редактора кода  
.git            файл настроек git  
.gitignore      файл настроек git-игнора  
Gruntfile.js    файл настроек Grunt  
gulpfile.js     файл настроек gulp  
package.json    файл с информацией о проекте и зависимостями        
README.md       файл с информацией о проекте
4. Создаём под-директории проекта 
src/
  ../_design     файлы дизайна
  ../coffee      файлы coffee       
  ../css         файлы css
  ../docs        документы, прочие файлы
  ../fonts       шрифты
  ../img         изображения для дизайна
  ../js          js-скрипты
  ../js/libs     js-библиотеки
  ../less        файлы less
  ../sass        файлы sass
  ../pict        изображения для контента
  ../index.html  основной html-файл
5. Установка инструментов
  1. Bower, настройка .bowerrc
  2. Grunt или gulp + плагины
  3. Настройка Gruntfile.js или gulpfile.js
  4. Установка node-модулей
6. Установка библиотек bower i jquery react angular ...
