Мой Справочник | node.js - Начало работы

Начало работы

node

 node

Запуск скриптов:
--------------------------
node some-script.js 		Режим node в консоли. Можно писать js-код или выполнять скрипты

npm

 Установка пакетов:
--------------------------
npm install /package-name + flag/
npm i /package-name + flag/

Удаление пакетов:
--------------------------
npm uninstall /package-name + flag/

Установка всех пакетов
(если есть готовый package.json):
--------------------------
npm install 		node package manager - пакетный менеджер для node

Флаги

 
-g, --global  // global
--save          // save for work
--save-dev    // save for development
-y               // yes for npm init
 Все установленные пакеты с save будут сохранены в файл package.json

Инициализация проекта

 mkdir some-dir
cd some-dir
npm init || npm init -y 		Так создаются проекты с нуля, в других случаях можно брать уже готовый файл package.json

package.json

 Создаётся при инициализации. Все пакеты с флагами --save и --save-dev будут записаны.
