Модальное окно

ЛИЧНЫЙ КАБИНЕТ

ВВЕДИТЕ ПОЖАЛУЙСТА СВОЙ ЛОГИН И ПАРОЛЬ
ЕСЛИ ВЫ ЗАБЫЛИ ПАРОЛЬ
Я ЗАБЫЛ ПАРОЛЬ

Показать\убрать окно

.modal-window-show {display: block;}
<a class="modal-window-btn">Модальное окно</a>
    ...
<div class="modal-window"> <!--CSS {display: none;}--> 
    ...
    <input type="submit" value="ВОЙТИ"> 
    <div class="close"></div>            
</div>

JS-код:

var modalWindowBtn = document.querySelector(".modal-window-btn");
var modalWindow = document.querySelector(".modal-window");
var modalWindowClose = document.querySelector(".close");


modalWindowBtn.addEventListener('click', function(event){
    event.preventDefault(); // сбрасывает дефолтные настройки
    modalWindow.classList.add("modal-window-show");
}); 
modalWindowClose.addEventListener('click', function(){
    modalWindow.classList.remove("modal-window-show");
});

Если в браузере у посетителей отключится JS, нужно сделать так, чтобы окна могли открываться в отдельных html-страницах без JS.

Слой-маска для затемнения экрана

.overlay {
  display: none;
  width: 100%;
  height: 100%;
  background-color: rgba(0,0,0,.75);
  position: fixed;
  top: 0;
  left: 0;
  z-index: 888;
}

Стили для мод. окна

.modal {
  display: none;
  position: fixed;
  padding: 20px;
  width: 300px;
  margin-left: -150px;
  top: 100px;
  left: 50%;
  z-index: 900;
}

Скрыть вертикальный скролл у боди, при появлении мод. окна

body.open-modal {
  overflow-y: hidden;
  padding-right: 15px;
}
