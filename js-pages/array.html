Вызов массива
Когда большое количество данных, вместо большого количества переменных удобнее создавать массивы var array = [0, 1, 2, 3];
//создали массив с 4-мя значениями
//форма записи строчного массива
var array = [
    'Mersedes',
    'BMW',
    'Audi',
    'Porshe'
]
alert(array);//покажет все элементы в массиве
var arr = [];//пустой массив 		Массив вызывается также как и переменные, ключевым словом var. Значения массива записываются в [ ] через запятую. Массивы по типу объекты, поэтому синтаксис в добавлении\удалении элементов через точку.
Также можно создавать массивы с помощью функции new Array var myArr = new Array(1,2,3);
console.log(myArr); //[1, 2, 3]
Проверка на массив var a = [ ];
Array.isArray(a); // true
Индексы значений массива
У каждого элемента массива есть свой индекс, индексируются элементы с 0 var array = [
    'Mersedes', //0-й индекс
    'BMW',      //1-й индекс
    'Audi',     //2-й индекс
    'Porshe'    //3-й индекс
] 		Можно изменять значения в массиве по индексу:
array[0] = 'Bugatti';
array[1] = 'Alfa Romeo';
array[2] = 'Lamborghini';
array[3] = 'Ferrari';
 или добалять новые:
array[4] = 'Maserati';
Индексы не обязательно могут идти друг за другом, могут быть незаполненные ([0], [1], [3], [10])
length
Можно узнать о количестве элементов в массиве console.log(array.length,); //4
console.log(array[array.length-1]); //покажет значение последнего элемента в массиве
array[array.length] = 'Fiat'; //аналог aray[4] = 'Fiat'; добавит новый элемент в конец массива 		length всегда содержит число элементов в массиве
С помощью length можно добавлять новый элемент в массив, т.к. общее число элементов в массиве всегда равно по индексу следующему новому элементу массива.
На самом деле свойство length показывает не число элементов в массиве, а индекс последнего элемента + 1
push
Добавляет элементы в КОНЕЦ массива array.push('De Tomaso', 'IVECO', 'Lancia'); Добавяться 3 новых элемента в КОНЕЦ массива array
unshift
Добавляет элементы в НАЧАЛО массива array.unshift('De Tomaso', 'IVECO', 'Lancia'); Добавяться 3 новых элемента в НАЧАЛО массива array
pop
Удаляет элементы с КОНЦА массива array.pop(); Удаляет последний элемент из массива array
Можно присваивать значение переменной удалёный элемент массива var removeItem = array.pop();
shift
Удаляет элементы с НАЧАЛА массива array.shift(); Удаляет ПЕРВЫЙ элемент из массива array
join
Объединяет все элементы массива в одну строку array.join(); Можно вставлять в ('*') любые разделяющие символы
sort
Cортирует элементы массива array.sort();
reverse
Меняет порядок элемента массива на реверсивный array.reverse();
concat
Объединяет массивы var myArr1 = [1, 2, 3, 4];
var myArr2 = [5, 6, 7, 8];
console.log(myArr1.concat(myArr2));
// [1, 2, 3, 4, 5, 6, 7, 8] 		Метод склеит два массива в один, сохраняя порядок, не меняет исходный массив
slice
Вытаскивает элементы из массива var myArr = [1, 2, 3, 4];
console.log(myArr.slice(-2));
// [3, 4] 		Вытащит 2 последних элемента из массива, не меняет исходный массив
splice
Удаляет и заменяет элементы в массив, возвращает удалённые элементы var myArr = [1, 2, 3, 4];
console.log(myArr.splice(1, 1, 123));
// [2]
console.log(myArr);
// [1, 123, 3, 4] 		1 аргумент - индекс элементов, с которых нужно начать удаление,
2 - сколько элементов нужно удалить,
3 и последующие - элементы, которые нужно вставить на место удалённых
Многомерные массивы
Многомерные массивы - массивы, вложенные друг в друга var arr1 = [1, 2, 3],
    arr2 = [4, 5, 6],
    arr3 = [7, 8, 9];
var bigArr = [arr1, arr2, arr3];
// создали двумерный массив bigArr
Просмотр значений в многомерном массиве console.log(bigArr[2] [0]); // 7 Указываем несколько индексов друг за другом, в данном случае первый элемент третьего массива
