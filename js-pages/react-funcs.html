React.createElement()
Создание React-элемента // <h1>Some text</h1> var h1 = React.createElement(
  'h1',
   null,
  'Some text'
);
// <h1 class="some-class">Some text</h1> var h1 = React.createElement(
  'h1',
   {className: 'some-class'},
  'Some text'
); 		Принимает три параметра:
1 - какой элемент создавать;
2 - параметры элемента
3 - содержание элемента

Могут быть также различные переменные в третьем параметре (принимает сколь угодно значений)
ReactDOM.render()
Основная функция для построения виртуального DOM ReactDOM.render(
  h1,
  document.getElementById('content')
);


<div id="content"><h1>Some text</h1></div>
ReactDOM.render(
  React.createElement('div', null, h1,h1,h1),
  document.getElementById('content')
);


<div id="content">
  <h1>Some text</h1>
  <h1>Some text</h1>
  <h1>Some text</h1>
</div> 		Принимает два параметра:
1 - что рендерить;
ReactDOM принимает только один первый параметр на вход.

Можно также записывать React.createElement или переменную первым параметром

2 - куда рендерить;
id тега в html-разметке
setState()
Функция сравнения для state 
 // статичный state. username может быть у кого угодно
  state = {
    persons: [
      { username: 'Sergey' },
      { username: 'John' },
      { username: 'Olga' }
    ]
  }
// функция для изменения state при событии
  printName = (event) => {
    this.setState({
      persons: [
// будет изменено значение username 1 элемента массива
// оно будет равно значению в поле
        { username: event.target.value},
        { username: 'John' },
        { username: 'Olga' }
      ]
    })
  }
// навешивание обработчика событий
// при любом изменении в этот поле будет запускаться что-то, что
// лежит в свойстве changed
  <input
      type='text'
      style={style}
      onChange={props.changed}
  />

// а там лежит эта функция
<UserInput
  changed={this.printName}
/>
Сравнивает состояния state, можно реализовывать вместе с обытиями
jsx
Этот формат симбиоз js и html. В нём можно записывать html-код как и в html-файлах без кавычек. В JSX в {} можно записывать обычный js-код Транспилируется с помощью транспилятора babel
