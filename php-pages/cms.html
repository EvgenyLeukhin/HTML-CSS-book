Права записи\создания файлов

Заходим в cd ~/../../opt/lampp/htdocs/wordpress/

Посмотреть запущенных пользователей:

ps aux | grep httpd

Посмотреть кем созданы файлы или каталоги:

ll /path/

Нужно выдать права deamon:

chown -R deamon:deamon /wp-content/uploads/
... www-data:www-data ... (ngnix)
Fix upload images phpMyAdmin -> DB -> wp_options -> upload_path -> Edit -> option_value -> /opt/lampp/htdocs/wordpress/wp-content/uploads/ -> Вперёд
