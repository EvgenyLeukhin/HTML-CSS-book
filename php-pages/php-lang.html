Мой Справочник | PHP | Язык PHP

Синтаксис 
<?php
  ...php-code...
?>
// строчный комментарий
/* ... */ многострочный комментарий
# комментарий в unix-системах
 Язык PHP - скриптовый, служит для создания динамических веб-страниц, работает на стороне сервера, в браузере php-код не отображается.

Слово php в открывающем теге можно опустить

Если после php-кода нет html-кода, то ?> можно опустить.
Локальный сервер
  • C:/Server/OpenServer/domains - рабочий каталог OpenServer
  • C:/xampp - рабочий каталог xampp
  • C:/WebServers/home - рабочий каталог Denwer
 Для разработки на языке php необходимо иметь локальный сервер + интерпретатор языка php + базы данных. Лучше всего использовать для разработки уже собраный пакет программ, таких как Open Server, xammp, Denwer
