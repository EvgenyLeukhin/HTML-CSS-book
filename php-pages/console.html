Настройки консоли
1. Установка chocolatey Запустить командную строку от администратора в Windows и ввести:
@powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin" 		Пакетный менеджер консоли в Windows
2. Установка babun choco install babun -y

Как пользоваться:

Контекстное меню - Babun here или команда в консоли babun 		Консольный интерпретатор php. Можно установить как через сайт, так и через консоль. Получаем среду как в Linux
3. Установка пакетов для PHP choco install -y vcredist2015 //библ. языка
choco install -y php atom //поддержка в атоме
4. Запуск PHP-кода
(файл index.php) 		Встаём в директорию с файлом index.php
php index.php
index.php (php-код)
<?php

echo 2 ** 10; // 2 в 10 степени 		Язык PHP при разработке может выполняться исключительно в консоли
