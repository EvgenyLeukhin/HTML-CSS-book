Консоль
1. Установка chocolatey Запустить командную строку от администратора в Windows и ввести: 
@powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"
 Пакетный менеджер консоли в Windows
2. Установка babun 
choco install babun -y
Как пользоваться:
Контекстное меню -> Babun here или команда в консоли babun		 Консоль для PHP-кода
3. Установка пакетов для PHP 
choco install -y vcredist2015      // библ. языка
choco install -y php atom          // поддержка в атоме
Как пользоваться: 
Встаём в директорию с файлом index.php -> php название php-файла
 Интерпретатор PHP-кода в консоли babun

Язык PHP при разработке может выполняться исключительно в консоли
Консольные команды
IP-адрес клиента 
ipconfig
 Узнать свой IP-адрес. IP-адрес присваивается компьютеру при входе в интернет - серверная технология DHCP провайдера
IP-адрес сервера 
nslookup адрес сайта или его IP
Проверка доступности 
ping адрес сайта или его IP
Трассировка 
tracert адрес сайта или его IP
